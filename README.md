# Hybrid Search Workshop

## Getting Started

### 1. Open Kibana

Navigate to the Kibana server in your browser:

**https://genworthworkshop-005c3c.kb.eastus2.azure.elastic-cloud.com**

### 2. Log In

- **Username:** `workshop`
- **Password:** *(provided by instructor)*

### 3. Open Dev Tools

Once logged in, click the hamburger menu (top left) and navigate to **Management > Dev Tools**, or use the search bar at the top to search for "Dev Tools".

## Workshop Files

There are two versions of the student exercises:

| File | Description |
|------|-------------|
| **student.md** | Full guide with detailed explanations before each exercise. Read this to understand *what* each query does and *why*. |
| **student_all.md** | All 20 exercises as bare commands, ready to paste into Dev Tools in one go. Use this to save time if you're comfortable with the concepts. |

Both files contain the same queries. Pick whichever suits your pace.
